## Como Usar

### HTML
```html
<img src="https://raw.githubusercontent.com/djotabr/icones_operadoras/main/logos/VIVO.png" alt="VIVO">
```

### Markdown
```markdown
![VIVO](https://raw.githubusercontent.com/djotabr/icones_operadoras/main/logos/VIVO.png)
```

### CSS (Background)
```css
.operadora-vivo {
  background-image: url('https://raw.githubusercontent.com/djotabr/icones_operadoras/main/logos/VIVO.png');
}
```

Substitua `VIVO` pelo nome da operadora desejada (mantendo o case-sensitive).
